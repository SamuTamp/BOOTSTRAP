Bootstrap è un framework CSS e JS che serve per facilitare lo sviluppo di pagine web responsive, ovvero che si adattano a schermi di diverse dimensioni (desktop, tablet, smartphone). Con Bootstrap puoi: Creare layout con griglie (grid system). Usare componenti pronti (bottoni, navbar, card, modali). Applicare stili coerenti senza scrivere CSS da zero. CDN (Content Delivery Network) è una rete di server distribuiti nel mondo che serve file (come CSS, JS, immagini) più velocemente agli utenti. Vantaggi rispetto a scaricare Bootstrap sul sito: Velocità: i file vengono caricati dal server più vicino all’utente. Caching: se un utente ha già scaricato lo stesso file da un altro sito, il browser lo riutilizza senza scaricarlo di nuovo. Aggiornamenti facili: basta cambiare il link del CDN per usare l’ultima versione. Svantaggio: dipendenza da una connessione esterna. Serve a rendere la pagina responsive. width=device-width: la larghezza della pagina si adatta alla larghezza del dispositivo. initial-scale=1: la pagina parte con zoom iniziale 100%. Senza questo meta tag, le pagine su mobile potrebbero apparire zoomate o non scalate correttamente. Differenza tra Container e Container Fluid container: ha larghezza massima fissa e si adatta a breakpoint (sm, md, lg, xl). container-fluid: occupa sempre 100% della larghezza dello schermo, indipendentemente dalla dimensione del dispositivo.
